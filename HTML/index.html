HTML

: h1태그 내부에 style="color:teal; text-align:center"을 넣어 CSS스타일 설정 가능 바꾸었다. : 색상은 color:teal 아니면 color:rgb(0,128,128), 그리고 color:#008080 으로 사용할 수 있다.

HTML은 HyperText Markup Language의 약자로써 웹 페이지의 전체적인 골격을 담당한다.
다양한 태그들로 구성되고, 각 태그는 웹 페이지의 디자인이나 기능을 결정하는데 사용된다.
또한, 각 태그에는 attribute(속성)이 존재하며, 속성은 태그의 추가적인 정보를 제공한다.

목차

  1. 태그의 종류
  2. 엔티티
  3. 문자셋
  4. 링크

태그의 종류

  • 제목 : <hn> : n은 1~6까지의 자연수이다. 숫자가 적을수록 더 크다.
  • 단락 : <p>
  • 개행 : <br> : 종료 태그가 없는 빈 태그이다.
  • 텍스트 서식 그대로 : <pre> : 태그 내부의 내용은 모든 띄어쓰기와 줄 바꿈이 그대로 표현된다.
  • 수평선 : <hr>
  • 강조 : <b> or <strong> : bold 태그는 그저 텍스트를 굵게 표현해 줄 뿐이지만 strong태그는 내부의 내용도 중요하다는 의미를 가진다(이 뒤부터는 태그가 두 가지 서술될 경우 전자는 그냥 태그, 후자를 의미를 가지는 태그로 서술할 것이다).
  • 이탤릭체(기울임) : <i> or <em>
  • 하이라이팅 : <mark>
  • 줄 그음 : <del>
  • 밑줄 : <ins>
  • 위첨자 : <sup>
  • 아래첨자 : <sub>
  • 인용구 : <q> : 앞뒤에 큰따옴표가 붙는다.
  • 블록 인용구 : <blockquote> : 따옴표는 붙지 않고, 그저 내용을 별도의 단락으로 구분한다.
  • 부가설명(?) : <abbr> : 나X위키의 주석과 같이 해당 태그의 내용에 마우스를 올릴 경우, title속성에 명시한 내용이 나타난다.
  • 주소 : <address> : 말 그대로 주소를 나타내기 위한 (쓸모 없을 것 같은)태그다. 위아래로 약간의 공백이 생성되고, 내부 내용은 이탤릭체로 표현된다.

엔티티

HTML에는 미리 예약된 몇몇 문자가 있으며, 이러한 문자를 예약어라고 한다. 예약어를 HTML코드에서 사용하면 웹 브라우저는 평소와는 다른 뜻으로 해석한다.
그래서 예약어를 사용하기 위해 별도로 만든 문자셋을 엔티티(Entity)라고 한다. 엔티티의 형태로는,

  • &엔티티이름;
  • &#엔티티숫자;
등이 있다.

대표적인 엔티티

  • 줄바꿈 없는 공백 #160 or nbsp
  • 부등호(보다 작은) #60 or lt
  • 부등호(보다 큰) #62 or gt
  • and기호 #38 or amp
  • 큰따옴표 #34 or quot
  • 작은따옴표 #39 or apos

발음 구별 부호 엔티티

  • #768 ex) à
  • #769 ex) á
  • #770 ex) â
  • #771 ex) ã

특수 문자 엔티티

  • 센트 #162 or cent
  • 파운드화 #163 or pound
  • 엔화 #165 or yen
  • 유로화 #8364 or euro
  • 저작권 #169 or copy
  • 등록상표 #174 or reg
  • 곱셈 #215 or times
  • 나눗셈 #247 or divide

문자셋

웹 브라우저가 HTML문서를 정확하게 나타내기 위해서 해당 문서가 어떤 문자셋(Character Set)으로 저장되었는지 알아야 한다.
따라서 HTML문서가 저장될 때 사용된 문자셋에 대한 정보를 <head>태그 내의 <meta>태그에 명시한다.

대표적인 문자셋

  • ASCII : 가장 처음 만들어진 문자셋으로, 인터넷에서 사용할 수 있는 127개의 영문자와 숫자로 이루어져 있다.
  • ANSI : 윈도우즈에서 만든 문자셋으로, 총 256개의 문자 코드를 지원한다.
  • ISO-8859-1 : 256개의 문자 코드를 지원하는 HTML4의 기본 문자셋이다.
  • UTF-8 : 세상에 있는 거의 모든 문자를 표현할 수 있는 유니코드 문자를 지원하는 HTML5의 기본 문자셋이다.

링크

HTML에서는 다른 페이지나 사이트로 이동할 수 있는 수많은 하이퍼링크(Hyperlink)가 존재한다.
이러한 링크를 간단히 링크(Link)라고도 부르며, HTML에서는 <a>태그로 표현한다.
a 태그는 텍스트나 단락, 이미지 등 다양한 HTML 요소에 사용할 수 있다.

href 속성

a 태그의 href속성은 연결할 페이지나 사이트의 URL주소를 명시한다.
ex)

구글입니다.

 여기서 href속성값으로는 https://www.google.com, a태그 내부내용으론 '구글입니다.'라는 문장을 사용했다.

target 속성

target 속성은 링크로 연결된 문서를 어디에서 열지 명시한다.
  • _blank : 링크로 연결된 문서를 새 창이나 새 탭에서 연다.
  • _self : 링크로 연결된 문서를 현재 프레임(Frame)에서 연다(기본설정(Default)).
  • _parent : 링크로 연결된 문서를 부모 프레임에서 연다.
  • _top : 링크로 연결된 문서를 현재 창의 가장 상위 프레임에서 연다.
  • 프레임 이름 : 링크로 연결된 문서를 지정된 프레임에서 연다.
마지막 프레임 이름 ex) 
코드 : 
<h2><a href="index.html" target="myframe">Hello Again!!</a></h2>
<iframe name="myframe" style="width:50%; height:330px"></iframe>

Hello Again!!

링크의 상태(state)

  • link : 아직 한번도 방문한 적이 없는 상태(Default)
  • visited : 한 번이라도 방문한 적이 있는 상태
  • hover : 링크 위에 마우스를 올려놓은 상태
  • active : 링크를 마우스로 누르고 있는 상태

기본적으로 링크가 걸린 텍스트는 밑줄에, 텍스트 색상이 파란색으로 변경된다.
visited 상태의 링크는 밑줄에, 텍스트 색상이 보라색으로 변경된다.
active 상태의 링크는 밑줄에, 텍스트 색상이 빨간색으로 변경된다.
